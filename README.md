# Interactive-Dashboard
An interactive web-based dashboard developed using R Shiny. The dashboard dynamically visualizes the Kenyan human gut virome.

The R-script and dataset used in building the dashboard can be found in the folder (https://github.com/JimmySmilez/Interactive-Dashboard/tree/8a2e38c9f24e4d3395ca24df7d61d3e5df775443/dashboard) on this repo.
