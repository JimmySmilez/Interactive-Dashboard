# Interactive-Dashboard
An interactive web-based dashboard developed using R Shiny. The dashboard dynamically visualizes the Kenyan human gut virome.

The R-script and dataset used in building the dashboard can be found in the folder [dashboard](https://github.com/JimmySmilez/Interactive-Dashboard/tree/main/dashboard) on this repo.
